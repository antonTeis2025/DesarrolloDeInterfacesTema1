Primero, creamos una tabla clientes con estos campos:

```sql
CREATE TABLE "clientes" (
	"codigo"	INTEGER,
	"dni"	TEXT NOT NULL UNIQUE,
	"apellidos"	TEXT NOT NULL,
	"nombre"	TEXT NOT NULL,
	"provincia"	TEXT,
	"sexo"	TEXT,
	"fechaalta"	TEXT,
	"formaspago"	TEXT,
	PRIMARY KEY("codigo" AUTOINCREMENT)
);
```

---
Para insertar datos usaremos la clase QtSql. En la clase Conexion, definiremos un método estatico (debe serlo para pasarle argumentos al parecer) que inserte los datos de un cliente en BBDD.

Como se había comentado, los **clientes se referencian como un diccionario** de python con esta estructura:

`['DNI', 'Apellidos', 'Nombre', 'Fecha', 'Provincia', 'Genero', ['metodos', 'pago']]
	Los metodos de pago son una lista porque pueden ser varios

El método estático convertirá y formateará este diccionario a una consulta SQL de esta manera:

```python
class Conexion():  
    @staticmethod  
    def insertarCliente(cliente):  
        query = QtSql.QSqlQuery()  
        # Preparamos la query  
        query.prepare("INSERT INTO clientes (dni, apellidos, nombre, fechaalta, provincia, sexo, formaspago)"  
                      " VALUES (:dni, :apellidos, :nombre, :fechaalta, :provincia, :sexo, :formaspago)")  
        # Sustituimos los valores con bindValue() para evitar SQLi  
        query.bindValue(':dni', str(cliente[0]))  
        query.bindValue(':apellidos', str(cliente[1]))  
        query.bindValue(':nombre', str(cliente[2]))  
        query.bindValue(':fechaalta', str(cliente[3]))  
        query.bindValue(':provincia', str(cliente[4]))  
        query.bindValue(':sexo', str(cliente[5]))  
        query.bindValue(':formaspago', str(cliente[6]))  
		  
        if query.exec():  
            print("[+] Insercion correcta de ", cliente)  
        else:  
            print("Error ", query.lastError().text())
```
