Para crear un dialogo de seleccion de fichero, que es nativo de Windows, tenemos que crear una clase extendida de **QFileDialog**.

```python
class DialogoArchivo(QtWidgets.QFileDialog):  
    def __init__(self, parent=None):  
        super(DialogoArchivo, self).__init__(parent)
```

La instanciamos desde main con una variable global

```python
# Instanciamos el dialogo de abrir archivo  
var.dlgAbrir = DialogoArchivo()
```

Para mostrarlo: 

```python
var.dlgAbrir.show()
```
